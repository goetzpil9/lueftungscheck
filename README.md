# Lüftungscheck -> https://goetzpil9.github.io/lueftungscheck/

Zeigt auf einen Blick, ob Lüften gerade sinnvoll ist – oder ob es die Luftfeuchtigkeit in der Wohnung eher verschlimmert.

## Wie es funktioniert

Temperatur und Luftfeuchtigkeit innen eintragen, die Außenwerte holt sich die App automatisch per Postleitzahl. Verglichen wird der **Taupunkt** beider Seiten – die Temperatur, bei der Luft anfängt zu kondensieren. Ist der Taupunkt draußen niedriger als drinnen, lohnt sich Lüften.

## Features

- 📍 Automatischer Wetterabruf per PLZ
- 📅 48-Stunden-Prognose mit den günstigsten Lüftungsfenstern
- 📴 Funktioniert offline (installierbar als App)
- 🔒 Keine Werbung, kein Tracking, kein Login

## Technologie

- Wetterdaten: [Open-Meteo](https://open-meteo.com)
- Physik: Magnus-Formel für Taupunkt und absolute Feuchte
- Progressive Web App (PWA) – auf dem Smartphone installierbar
